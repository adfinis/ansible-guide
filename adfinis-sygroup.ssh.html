Default valuesÔÉÅ

A list of all host key files (rsa and ed25519 are recommended). Default is only rsa; ed25519 is recommended if supported by sshd.

ssh_host_keys:
  - /etc/ssh/ssh_host_rsa_key
  - /etc/ssh/ssh_host_ed25519_key

A list of all ports for sshd to listen on.

ssh_daemon_ports:
  - 22

Allow root login. choices:

  • yes (password and publickey)

  • no (not allowed)

  • without-password (only allowed with publickey)

ssh_daemon_permit_root_login: 'yes'

Allow password authentication.

ssh_daemon_password_authentication: 'yes'

Enable or disable the sshd X11 forwarding.

ssh_daemon_x11_forwarding: 'no'

Environment variables to accept by sshd.

ssh_daemon_accept_env:
  - 'LANG'
  - 'LC_*'

A list of places to search for authorized keys files. CentOS 6 doesn‚Äôt support multiple places, so default is just $HOME/.ssh/authorized_keys.

ssh_daemon_authorized_keys:
  - '/etc/ssh/authorized_keys/%u'
  - '%h/.ssh/authorized_keys'

A list of ciphers used by sshd.

ssh_daemon_ciphers:
  - chacha20-poly1305@openssh.com
  - aes256-gcm@openssh.com
  - aes128-gcm@openssh.com
  - aes256-ctr
  - aes192-ctr
  - aes128-ctr

A list of KexAlgorithms used by sshd.

ssh_daemon_kex_algorithms:
  - curve25519-sha256@libssh.org
  - diffie-hellman-group-exchange-sha256

A list of MACs used by sshd.

ssh_daemon_macs:
  - hmac-sha2-512-etm@openssh.com
  - hmac-sha2-256-etm@openssh.com
  - umac-128-etm@openssh.com

Host key algorithms.

ssh_host_key_algorithms:
  - ssh-ed25519-cert-v01@openssh.com
  - ssh-rsa-cert-v01@openssh.com
  - ssh-ed25519
  - ssh-rsa
